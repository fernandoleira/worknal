# Worknal

Web app to keep track of important daily entries and goals.

## Getting Started

```bash
# Project setup
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```
